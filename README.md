# nexus

Design: [Figma design][figma-design]

Template: [Nexus - AI Event Landing Page][figma-community]

A modern and structured landing page template for AI summits, conferences, and
tech events. Designed with a clean layout, it includes key sections like the
agenda, speakers, and registration while keeping everything flexible and easy to
customize.

This template is built in Figma with a focus on clarity and usability, so you can
quickly adapt it to different event types. The design keeps things minimal yet
engaging, making it a solid starting point for your next project.
